Compiler Help Guide

Important Notes

• All operations must use parentheses
• Arguments must be interger numbers
• The input should not follow ' ; '
• Division by zero will return an error

This compiler supports the following mathematical operations:

Binary Operations (two arguments)

add(num1, num2) - Addition
Adds two numbers together. Example: add(5, 3) returns 8
subtract(num1, num2) - Subtraction
Subtracts num2 from num1. Example: subtract(10, 4) returns 6
multiply(num1, num2) - Multiplication
Multiplies two numbers. Example: multiply(3, 7) returns 21
divide(num1, num2) - Division
Divides num1 by num2. Example: divide(15, 3) returns 5
mod(num1, num2) - Modulus
Returns the remainder after division. Example: mod(10, 3) returns 1
power(num1, num2) - Exponentiation
Raises num1 to the power of num2. Example: power(2, 4) returns 16

Unary Operations (one argument)

iseven(num) - Even number check
Returns true if the number is divisible by 2. Example: iseven(8) returns true
isodd(num) - Odd number check
Returns true if the number is not divisible by 2. Example: isodd(7) returns true

Mathematical Functions

sqrt(num) - Square root
Calculates the square root of a number. Example: sqrt(16) returns 4

Geometric Calculations

areaCircle(radius) - Area of circle
Calculates πr². Example: areaCircle(5) returns ~78.54
circumferenceCircle(radius) - Circumference of circle
Calculates 2πr. Example: circumferenceCircle(5) returns ~31.42
areaRectangle(length, width) - Area of rectangle
Calculates length × width. Example: areaRectangle(4, 5) returns 20
perimeterRectangle(length, width) - Perimeter of rectangle
Calculates 2×(length + width). Example: perimeterRectangle(4, 5) returns 18
areaSquare(side) - Area of square
Calculates side². Example: areaSquare(5) returns 25
perimeterSquare(side) - Perimeter of square
Calculates 4×side. Example: perimeterSquare(5) returns 20
areaTriangle(base, height) - Area of triangle
Calculates ½×base×height. Example: areaTriangle(6, 4) returns 12
perimeterTriangle(a, b, c) - Perimeter of triangle
Calculates a + b + c. Example: perimeterTriangle(3, 4, 5) returns 12

Mathematical Functions

factorial(n) - Factorial of a number
Calculates n!. Example: factorial(5) returns 120
isPrime(n) - Check if number is prime
Returns true if number is prime. Example: isPrime(7) returns true
gcd(a, b) - Greatest common divisor
Calculates GCD of two numbers. Example: gcd(12, 18) returns 6
lcm(a, b) - Least common multiple
Calculates LCM of two numbers. Example: lcm(12, 18) returns 36
average(a, b) - Average of two numbers
Calculates (a + b)/2. Example: average(4, 6) returns 5
maxOfTwo(a, b) - Maximum of two numbers
Returns the larger number. Example: maxOfTwo(3, 5) returns 5
minOfTwo(a, b) - Minimum of two numbers
Returns the smaller number. Example: minOfTwo(3, 5) returns 3
absolute(a) - Absolute value
Returns non-negative value. Example: absolute(-5) returns 5
remainder(a, b) - Remainder after division
Same as mod. Example: remainder(10, 3) returns 1
return(a) - Returns the value
Simply returns the input. Example: return(5) returns 5

Unit Conversion

toRadian(degree) - Degrees to radians
Converts degrees to radians. Example: toRadian(180) returns π
toDegree(radian) - Radians to degrees
Converts radians to degrees. Example: toDegree(π) returns 180
toFahrenheit(celsius) - Celsius to Fahrenheit
Converts temperature. Example: toFahrenheit(0) returns 32
toCelsius(fahrenheit) - Fahrenheit to Celsius
Converts temperature. Example: toCelsius(32) returns 0
toMile(km) - Kilometers to miles
Converts distance. Example: toMile(1.60934) returns ~1
toKm(mile) - Miles to kilometers
Converts distance. Example: toKm(1) returns ~1.60934
toHour(minute) - Minutes to hours
Converts time. Example: toHour(120) returns 2
toMinute(hour) - Hours to minutes
Converts time. Example: toMinute(2) returns 120
toSecond(minute) - Minutes to seconds
Converts time. Example: toSecond(1) returns 60
toMinute(second) - Seconds to minutes
Converts time. Example: toMinute(60) returns 1

Additional Geometric Calculations

surfaceAreaCuboid(length, width, height) - Surface area of cuboid
Calculates 2(lw + lh + wh). Example: surfaceAreaCuboid(3, 4, 5) returns 94
volumeCone(radius, height) - Volume of cone
Calculates (1/3)πr²h. Example: volumeCone(3, 5) returns ~47.12
surfaceAreaCylinder(radius, height) - Surface area of cylinder
Calculates 2πrh + 2πr². Example: surfaceAreaCylinder(3, 5) returns ~150.8
percentage(total, part) - Percentage calculation
Calculates (part/total)*100. Example: percentage(200, 50) returns 25

Usage Examples

add(9, 11) → 20
Calculates and then adds them: 9+11=20
iseven(15) → false
Checks if 15 is even by 15%2 !==0 (false)
power(2, 3) → 8
Calculates 2³ and then gives result (8)
